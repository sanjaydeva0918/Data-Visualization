# Data Cleaning and Preprocessing Project

## Overview

This project focuses on cleaning and preprocessing a Google Play Store Reviews dataset using Python and Pandas.

## Objective

Prepare raw review data for analysis by handling missing values, removing duplicate records, standardizing formats, and validating data quality.

## Tools Used

* Python
* Pandas
* VS Code

## Dataset Information

The dataset contains user reviews, ratings, app versions, and review details collected from the Google Play Store.

## Data Cleaning Steps

* Removed duplicate records
* Handled missing values
* Converted date columns into datetime format
* Standardized column names
* Verified data types
* Generated statistical summaries

## Results

* Original Records: 151,559
* Cleaned Records: 151,272
* Missing Values Remaining: 0
* Average Rating: 2.87

## Output Files

* Raw Dataset
* Cleaned Dataset (google\_reviews\_cleaned.csv)
* Python Script
* Screenshots
* Summary Report
